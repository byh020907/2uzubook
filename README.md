# 2uzubook

## 소개

2uzubook은 2017년도 2학년 소프트웨어과 자바 팀 프로젝트로 기획된 웹 애플리케이션이다.

대덕소프트웨어마이스터고등학교 재학생들의 취업을 돕기 위해서 학생들의 레주메를 보다 접근성 있게 기업에 공개할 수 있는 소통의 창구를 마련하고자 기획하였다.
