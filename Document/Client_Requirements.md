* 로그인
    * user 테이블의 id와 password를 이용
    * 성공 시 세션 발급
    * 실패 시 메시지 알림

    * 세션 폐기
    * 홈 화면으로 복귀

* 회원가입
    * user 테이블 insert
    * 성공 시 로그인 후 홈 화면
    * 실패 시 메시지 알림 (ID 중복 등)

* 레주메 내용 추가
    * 분야 별 레주메 내용 추가
    * 성공 시 분야별 테이블 내용 추가
    * 실패 시 메시지 알림

* 레주메 내용 수정
    * 분야 별 레주메 내용 수정

* 레주메 내용 삭제
    * 분야 별 레주메 내용 삭제

* 서비스 등록 신청
    * company 테이블 insert

* 기업 로그인
    * company 테이블 key 이용
    * 성공 시 세션 발급
    * 실패 시 메시지 알림
